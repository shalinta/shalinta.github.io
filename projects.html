  • 1. e-voting dApp
    One of my notable projects showcased here is an e-voting decentralized application developed using Solidity, a language for writing smart contracts on the Ethereum blockchain. Leveraging the Truffle framework and Ganache for local blockchain development, I engineered a secure and transparent voting platform. By storing votes directly on the blockchain, this solution ensures the integrity of the electoral process, guaranteeing that votes remain tamper-proof and immutable. Through this project, I demonstrated my proficiency in blockchain technology and my commitment to developing innovative solutions for real-world challenges.
  • 1. Skincare Recommendation System
    I'm excited to share my project on a skincare recommendation system, which utilizes collaborative filtering techniques to offer personalized skincare suggestions. By analyzing user preferences and behaviors, this system generates tailored recommendations, ensuring users receive skincare products that best suit their individual needs. I implemented the recommendation engine using collaborative filtering algorithms, harnessing the power of data to deliver accurate and effective suggestions. To enhance user interaction and accessibility, I developed the frontend using Streamlit, a user-friendly framework for building interactive web applications with Python. Through this project, I aimed to simplify the skincare selection process and empower users to make informed decisions about their skincare routine.
